----------STRING operations continue
-------->Using LOWER() or UPPER()makes the comparison consistent.

1. UPPER(): Converts all letters in a string to uppercase (capital letters).
Syntax
UPPER(column_name);
EX:
select upper("Hello");   O/P:HELLO
select upper("heLLO World");

2. LOWER(): Converts all letters in a string to lowercase (small letters).
Syntax
LOWER(column_name);
EX:select lower("HEllO"); O/P:hello

3.-- convert while entering the data
CREATE TABLE TEST(ID INT,AGE INT,EMAIL VARCHAR(20));

DESCRIBE TEST;

-- converts the email into lowercase while inserting itself, u can see in output, mail="jhon@gmail.com"
INSERT INTO TEST(ID,AGE,EMAIL)VALUES(1,25,LOWER("Jhon@gmail.com"));

SELECT * FROM TEST;

3. trim():removes spaces from the beginning and end of a string.
Syntax:
TRIM(string)

Example
SELECT TRIM('   Hello World   ');

---->TRIM() can also remove specific characters.

Syntax
TRIM(character FROM string)

Example:
SELECT TRIM('x' FROM 'xxxHello Worldxxx');  O/P: Hello World

-------------------------LTRIM() and RTRIM()
-- LTRIM()	Removes spaces from left side
SELECT LTRIM('   Hello');

-- RTRIM()	Removes spaces from right side
SELECT RTRIM('Hello   ');

4.REVERSE():string function used to reverse the characters of a string.
Syntax
REVERSE(string)

EX:
select reverse("Hello GM");

------>Example with Numbers
SELECT REVERSE('12345');

Output:54321
Even numbers are treated as text characters, so they reverse.

5. FORMAT() is used to format a number with commas and a specified number of decimal places.
Syntax
FORMAT(number, decimal_places)
number → the value to format
decimal_places → number of digits after the decimal point

EX:
SELECT FORMAT(1234567.8912, 2);

 O/P:  1,234,567.89(Commas are added for readability, Only 2 decimal places are kept)

BASIC FORMULA FOR DISCOUNT:
Total Price = ITEM_PRICE * ITEM_QTY
Discount Amount = Total Price * Discount_Percentage / 100
Final Price = Total Price - Discount Amount

-- CALCULATE DISCOUNT(10%)
SELECT ITEM_NAME,ITEM_PRICE,ITEM_QTY,
ITEM_PRICE * ITEM_QTY AS TOTAL_PRICE,
(ITEM_PRICE * ITEM_QTY) * 10 / 100 AS DISCOUNT,
(ITEM_PRICE * ITEM_QTY) - ((ITEM_PRICE * ITEM_QTY) * 10 / 100) AS FINAL_PRICE
FROM CART;
